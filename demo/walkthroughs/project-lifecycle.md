# Demo project lifecycle

1. Create `sample` in the dashboard.
2. Provision `git.divband.ir/demo/sample`.
3. Provision Kubernetes namespace `project-sample`.
4. Attach `sample.divband.ir`.
5. Deploy `demo/sample-site`.
6. Add a custom domain with TXT verification.
7. Ask the AI assistant to draft a text change.
8. Review the generated merge request.
