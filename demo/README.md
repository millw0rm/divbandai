# Demo space

This directory is for safe demo material that explains or simulates the divband platform before production integrations are live.

Use this space for:

- Example hosted sites.
- Walkthrough scripts — start with [`walkthroughs/project-lifecycle.md`](walkthroughs/project-lifecycle.md).
- Demo screenshots.
- Sample dashboard flows.
- Mock project definitions.

Demo content may be deployed only to explicit demo environments. It must not contain production secrets or real customer data.
