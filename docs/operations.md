# Operations

## Observability

Collect logs, metrics, and audit events with project and organization labels:

- `divband.io/project-id`
- `divband.io/project-slug`
- `divband.io/organization-id`
- `divband.io/environment`

## Alerts

Alert on:

- Failed namespace provisioning.
- Failed GitLab project creation.
- GitLab runner offline or unassigned.
- Failed deployment rollout.
- Certificate issuance failures.
- Domain verification failures.
- Namespace quota exhaustion.
- Repeated authentication failures.

## Audit events

Record:

- User login and role changes.
- Project creation, archive, and deletion.
- Domain add, verify, activate, and remove.
- GitLab project and runner creation.
- Secret create, update, and delete metadata.
- Deployment start, success, failure, and rollback.
- AI change request creation, approval, commit, and merge request creation.
